Complete las siguientes oraciones

  1. El elemento "hr" inserta una regla horizontal
  2. Para marcar un superíndice se utilza el elemento "sup" y para marcar un subíndice se utiliza el elemento "sub"
  3. El elemento de encabezado menos importante es "h6" y el elemento más importante es "h1"
  4. El elemento "ul" marca una lista desordenada
  5. El elemento "p" marca un párrafo
  6. El atríbuto type = "reset" en un elemento input inserta un botón que, cuando se presiona, restablece el contenido del formulario
  7. El elemento "tr" marca una fila de una tabla
  8. Los "Asteriscos" se usan por lo general como caracteres de enmarcado en un cuadro de contraseña
