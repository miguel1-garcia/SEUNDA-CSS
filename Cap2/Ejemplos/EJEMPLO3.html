Java How to Program, 9/e

De Java How Program 9/e

Características:

  • Extensa cobertura de los fundamentos: dos capítulos sobre instrucciones de control.
  • Enfoque ejemplos del mundo real
  • Conjunto de ejercicios Hacer la diferencia.
  • Introdución desde los primeros capítulos a las clases, objetos, métodos y cadenas.
  • Manejo de excepciones integrado.
  • Archivos, flujos y serialización de objetos
  • Selecciones modulare opcionales sobre el lenguaje y caractísticas de la biblioteco de la nueva versión Java SE 7.
  • Otros temas: Recursividad, búsqueda, ordenamiento, coleccioes genéricas, genéricos, estructuras de datos , applets, multimedias, multihilos, bases de datos/JDBC™, desarrollo de aplicaciones, servicios web y un caso de estudio opcional de Diseño orientado a objetos con un ATM.
